# Available widgets for the team dashboard page


## General widgets

### Hello widget
Shows a welcome message to the user.

### Todo-team widget
Shows a list of tasks for the current team.

### Todo-user widget
Shows a list of tasks for the current user.

### Boards widget
Shows a list of boards for the current team.

### Members widget
Shows a list of members for the current team. (Shows which members are online and which are offline.)
